02_자바스크립트_연산자

자바스크립트 연산자는 사용자가 조건식을 만들어서 원하는 결과를 출력하고자 할 때 사용하는 기호이다.

  • 산술연산자 : +, - , *. /. %(나머지)
  • 문자열 결합연산자 : +
  • 비교연산자 : >, <, ==, ===
  • 대입연산자 : =(오른쪽 값을 왼쪽에 대입), +=, -=
  • 증감연산자 : ++, --
  • 논리연산자 : &&(and) ||(파이프라인)(or) !(no)
  • 삼항조건연산자 : 조건식?실행문1:실행문2;(중요!)
  • 타입연산자 : typeof(데이터 타입을 구분해주는 연산자임)
