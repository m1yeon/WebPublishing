16_객체(object)

객체(object)란?
1. 이름(key)과 값(value)로 구성된 속성을 담아놓은 컨테이너이다
2. 객체에 변수를 선언하면 속성(property)을 사용할 수 있음. ex)document.getElementById('아이디명')
3. 객체에 함수를 선언하면 메서드(method)를 사용할 수 있음. ex)document.write('내용')

1. 객체의 종류

내장객체 : 자바스크립트에서 미리 정해놓은 객체로 다양한 종류가 있음

  • 수학객체(Math)
  • 날짜(date)객체
  • 배열(array)객체
  • 문자열(string)객체
  • 문서(document)객체
  • 윈도우(window)객체

사용자 정의 객체 : 내장 객체를 제외한 사용자가 원하는 내용을 토대로 직접 객체를 선언하여 작성하는 방법

1. 사용자 정의 객체 만들기_ let 객체이름 = new Object();

휴대폰 정보 입력하기 


    let phone = new Object();
    const btn = document.getElementById('order');

    btn.addEventListener('click',function(){
      const c = document.getElementById('company').value;
      const m = document.getElementById('model').value;
      const cl = document.getElementById('color').value;

      phone.company = c;
      phone.model = m;
      phone.color = cl;

      //2. r객체를 호출하여 내용 출력하기
      document.getElementById('result').innerHTML=` 출력 > 나의 핸드폰 제조회사는${phone.company}이고 모델명은${phone.model}이며, 색상은 ${phone.color}입니다`;
    });


2. 사용자 정의 객체 만들기 _ let 객체이름 = {이름 : 변수명(or 값)}

    
    let person ={
      firstName : '홍',
      lastName : '길동',
      age : '25',
      height:'175',
      weight:'70'
    }

두 번째 객체생성 방법을 통해 개인정보 입력양식을 만들어 사용자가 '성', '이름', '나이', '키', '몸무게'를 입력하고 '신청완료'를 누르면 객체정보가 출력되게 하기

나의 이름은 '홍길동'이고, 나이는 '23'세이며, 키는 '175', 몸무게는 '59'kg입니다.
