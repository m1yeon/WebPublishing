제어문(control)

제어문이란? 프로그램의 실행을 제어(control)하기 위해 사용하는 실행문으로서
1. 조건문(if, else, if else)
2. 선택문(switch case)
3. 반복문(while, do while, for)이 있다.

1. 조건문 _ if, else, if else

  • if : 조건식이 만족 할 때만 실행되는 조건문이다.
  • else : 조건식이 만족하면 true, 만족하지 않으면 false가 나오도록 하는 조건문이다.
  • else if : 조건식이 여러개일 경우 다양한 결과값을 얻기 위한 조건문이다.
