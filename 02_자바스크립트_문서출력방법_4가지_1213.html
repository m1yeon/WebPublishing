자바스크립트 문서출력방법 4가지

자바스크립트 언어를 통해 문서에 내용을 출력하는 방법 4가지에 대해 알아본다.

  1. document 객체에 wirte 메서드를 사용 - document.write('출력할 내용');
  2. window 객체를 활용항 ㅕ메세지 띄우기 - window.alert('출력할 내용');
  3. 브라우저 개발자모드의 console모드에서 내용 출력하기 - console.log('출력할 내용');
  4. document 객체의 innerHTML메서드로 내용 출력하기- document.getELement('id명').innerHTML="출력할 내용";
